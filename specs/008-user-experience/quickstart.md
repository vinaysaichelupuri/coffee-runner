# Quickstart: User Experience

## Dependencies
You will need to install the audio package:
```bash
npx expo install expo-av
```

## Running the App
1. Start the Metro bundler:
   ```bash
   npm run start
   ```
2. Navigate the application flow:
   - The app will start on the Home screen.
   - Tap "Play" to start the game loop.
   - Intentionally collide with an obstacle to trigger the "Game Over" screen.
   - Ensure the high score persists when hitting "Restart".

## Audio Testing
- Play the game on a physical device or emulator with audio enabled.
- Verify you hear sounds upon picking up coffee cups and hitting obstacles.
