# Quickstart: Obstacle System

To test the obstacle system once implemented:

1. Start the React Native bundler: `npm start`
2. Run on iOS or Android emulator.
3. The game should automatically start spawning obstacles at the top of the screen.
4. Verify that obstacles move downwards and disappear when they reach the bottom.
5. Check memory usage in the React Native debugger to ensure no leaks occur over time.
